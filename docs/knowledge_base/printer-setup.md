# Добавление сетевого принтера

Инструкция о том, как подключить рабочий компьютер к офисному принтеру и печатать по карте. Полезна новым сотрудникам и тем, кто переехал на другой этаж.

Все принтеры в офисе доступны через единый сервер печати `print.company.ru`. Печать защищённая: документ отправляется в общую очередь и распечатывается на любом принтере после прикладывания пропуска (печать по карте, follow-me).

## Пошаговая инструкция

1. Подключитесь к корпоративной сети — по Wi-Fi `CompanyCorp` или через VPN, если работаете удалённо.
2. На Windows откройте «Проводник» и в адресной строке введите `\\print.company.ru`. На macOS — «Настройки → Принтеры → Добавить → IP/Windows» и укажите тот же сервер.
3. В списке выберите принтер по нужному этажу (принтеры названы по схеме «этаж-зона», например `PRN-3-East`). Для печати по карте выберите общую очередь `SecurePrint`.
4. Согласитесь на установку драйвера — он подтянется с сервера автоматически. Если система спросит — выберите драйвер, предложенный по умолчанию.
5. Отправьте тестовую страницу. Документ встанет в общую очередь.
6. Подойдите к любому принтеру, приложите рабочий пропуск к считывателю и выберите ваши документы из списка для печати.

## Типичные проблемы

- print.company.ru не открывается → вы не в корпоративной сети; подключитесь к CompanyCorp или к VPN и повторите.
- Драйвер не ставится / нет прав → для установки могут понадобиться права администратора; заведите заявку на sd.company.ru, поддержка установит принтер удалённо.
- Карта (пропуск) не привязана к печати → при первом прикладывании система попросит ввести корпоративный логин и пароль — это разовая привязка пропуска.
- Документы не появляются на принтере → проверьте, что печатали в очередь SecurePrint, а не на конкретный принтер; задания в общей очереди хранятся ограниченное время.
- Принтер выдаёт ошибку (нет тонера, замятие) → используйте соседний принтер (печать follow-me работает на любом) и сообщите о неисправности в поддержку.
