# Установка корпоративного ПО

Инструкция о том, как поставить рабочие программы на корпоративный компьютер. Полезна всем сотрудникам — от установки браузера до запроса платной лицензии.

Большую часть одобренного ПО можно поставить самостоятельно через корпоративный портал приложений. Лицензионное и специализированное ПО требует отдельной заявки и согласования.

## Пошаговая инструкция

1. Откройте корпоративный портал приложений (ярлык «Центр приложений» / Software Center на рабочем столе или в меню «Пуск»).
2. Найдите нужную программу в каталоге и нажмите «Установить». Одобренное ПО ставится без прав администратора и без заявки.
3. Если программы нет в каталоге или она платная — откройте ServiceDesk на sd.company.ru и выберите услугу «Установка ПО / лицензия».
4. В заявке укажите название и версию ПО, для каких задач оно нужно и на какой компьютер ставить.
5. Отправьте заявку. Платное ПО проходит согласование руководителя (а иногда — бюджетного держателя), после чего выделяется лицензия.
6. Установку лицензионного ПО обычно выполняет поддержка удалённо. О готовности придёт письмо на @company.ru.

## Типичные проблемы

- «Недостаточно прав» при установке → ставьте через Центр приложений; ручная установка с правами администратора по умолчанию запрещена политикой безопасности.
- Нужной программы нет в каталоге → заведите заявку на sd.company.ru; если ПО одобрят, его добавят в каталог или установят вручную.
- Нужны права администратора для разработки/тестов → запросите временные права через ServiceDesk с обоснованием; их выдают на ограниченный срок.
- Лицензия закончилась, ПО просит активацию → не вводите личные или сторонние ключи; заведите заявку на продление лицензии.
- Хотите поставить личную/непроверенную программу → устанавливать неодобренное ПО на рабочую технику нельзя; сначала согласуйте через заявку.
