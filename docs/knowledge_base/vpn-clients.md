# VPN-клиенты под разные ОС

Справка о том, какой VPN-клиент ставить на ваше устройство и где его взять. Полезна, когда нужно подключиться к vpn.company.ru с непривычной операционной системы или с телефона.

Во всех случаях используется один клиент — GlobalProtect от Palo Alto Networks, портал подключения везде один: `vpn.company.ru`. Меняется только версия под платформу.

## Пошаговая инструкция

1. Откройте внутренний портал приложений (раздел «VPN / GlobalProtect»). Все официальные установщики лежат там — не качайте клиент со сторонних сайтов.
2. Выберите версию под вашу систему:
   - Windows 10/11 — GlobalProtect для Windows (64-bit), MSI-установщик.
   - macOS 12 и новее — GlobalProtect для macOS (universal, работает на Intel и Apple Silicon).
   - Linux (Ubuntu/RHEL) — GlobalProtect CLI или GUI-пакет; для дистрибутивов без пакета используйте openconnect с настройками от поддержки.
   - iOS / iPadOS — приложение «GlobalProtect» из App Store.
   - Android — приложение «GlobalProtect» из Google Play.
3. Установите клиент. На мобильных устройствах при первом запуске система попросит разрешить добавление VPN-конфигурации — согласитесь.
4. Введите портал `vpn.company.ru`, войдите по корпоративному логину и подтвердите MFA.

## Типичные проблемы

- На Linux нет официального GUI → используйте openconnect с протоколом GlobalProtect; конфиг и параметры выдаёт поддержка по заявке на sd.company.ru.
- Старая версия клиента не подключается → портал может требовать минимальную версию. Удалите старый клиент и поставьте актуальную сборку с портала.
- На личном телефоне не находится приложение → проверьте регион учётной записи в магазине приложений; при проблемах поддержка пришлёт прямую ссылку.
- На macOS с Apple Silicon клиент «не запускается» → убедитесь, что скачали universal-сборку, и разрешите системное расширение Palo Alto Networks в настройках безопасности.
